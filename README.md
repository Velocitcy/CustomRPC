## Main Components
- **2 Buttons** — configurable with labels and URLs  
- **State Text** — the subtext showing your current status  
- **Details Text** — the main description line  
- **Large Image** — shown with hover text  
- **Timestamps** — shows elapsed time since you started the activity  
- **Fully Customizable** through `index.ts`
